## 🧠 About This Repository

This repository documents my hands-on journey through the **AI Security Path on TryHackMe**.

With the rapid growth of AI systems like **LLMs (Large Language Models)** and automation tools, cybersecurity is evolving beyond traditional approaches. This path focuses on understanding how AI systems can be **attacked, exploited, and secured**.

---

## 🎯 Objectives

* Understand AI & Machine Learning fundamentals
* Identify vulnerabilities in AI systems
* Learn attack techniques such as:

  * Prompt Injection
  * Data Poisoning
  * Model Manipulation
* Explore defensive strategies for securing AI applications
* Build a strong foundation for **AI-focused cybersecurity roles**
